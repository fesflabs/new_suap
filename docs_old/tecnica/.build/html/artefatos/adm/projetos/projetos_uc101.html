Objetivo¶

Manter atualizada a lista de anexos previstos para os projetos de um edital.

Atores¶

Principais¶

Gerente sistêmico

Interessado¶

Não se aplica.

Pré-condições¶

Edital deve estar cadastrado.

Pós-condições¶

Não se aplica.

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado a partir do passo FA5.2.2 do caso de uso UC100 - Manter edital <v0.1> ao selecionar a opção EXTENSÃO > Projetos > Editais, escolhe um edital da listagem e escolhe a opção Visualizar, assim, ao visualizar o edital seleciona a opção Adicionar anexo

  2. O sistema apresenta a lista de anexos já cadastrados para o edital selecionado (RIN01)

  3. O ator seleciona a opção Adicionar anexo

  4. Na tela Adicionar anexo o ator preenche os campos (RIN02)

  5. O ator finaliza o caso de uso selecionando a opção Salvar

  6. O fluxo retorna para o ponto em que foi iniciado

Fluxo Alternativo¶

FA1 – Editar (FN-2)¶

  1. O ator seleciona a opção Editar de uma das linhas da listagem de anexos já cadastrados

  2. Na tela Editar anexo o ator altera os campos (RIN02), os dados do anexo já virão preenchidos

  3. O ator finaliza o caso de uso ao selecionar a opção Salvar

  4. O sistema apresenta a listagem do passo FN-2

FA2 – Remover (FN-2)¶

  1. O ator seleciona a opção Remover de uma das linhas da listagem de anexos já cadastrados

  2. O sistema apresenta a listagem do passo FN-2

FA3 – Anexar arquivo (FN-2)¶

  1. O ator seleciona a opção Anexar arquivo de uma das linhas da listagem de anexos já cadastrados

  2. Na tela Anexar arquivo o ator escolhe o arquivo a anexar

  3. O ator seleciona a opção Salvar

  4. O sistema apresenta a listagem do passo FN-2

Nota

Se possível anexar já ao incluir/editar.
