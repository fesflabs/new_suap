Descrição
As âncoras são links relativos à áreas dentro da página atual. Quando clicadas, a página deve se mover e mostrar o tópico definido.
Código HTML
<ul class="ancoras OPTIONAL_CLASS_PARENT">
    <li><a href="#id_1">Caixa de Entrada</a>
        <ul>
            <li><a href="#id_2">Aguardando recebimento</a></li>
            <li><a href="#id_3">Aguardando encaminhamento</a></li>
        </ul>
    </li>
    <li><a href="#id_4">Caixa de Saída</a></li>
</ul>
Tabela de Referência
Parâmetro Código Descrição
OPTIONAL_CLASS_PARENT .three-digits Exibe o marcador sempre com 3 dígitos.
Regras
As âncoras devem ser adicionadas sempre do block asideright.
