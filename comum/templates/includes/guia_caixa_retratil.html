Tabela de Referência
Parâmetro Código Descrição
OPTIONAL_CLASS .bordered Usado para exibir bordas englobando o conteúdo.
.collapsed Usado para que a caixa inicie fechada.
.filters Usado para caixas que possuam apenas filtros.
.help Usado para mostrar mensagens de ajuda.
.text Usado quando o conteúdo for somente texto em parágrafos.
Regras
É possível escrever variáveis em Python dentro do título da caixa, porém, deve-se, obrigatoriamente, manter as aspas.
