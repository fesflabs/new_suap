Finalidade¶

Permite o controle financeiro das atividades do setor.

Módulo Auxílios¶

Adicionar Auxílio¶

  1. Acesse Atividades Estudantis > Coordenação > Auxílios

  2. Clique em “Adicionar Auxílio”

  3. O sistema exibirá a seguinte tela para que o usuário informe os dados necessários:

  4. Clique em “Salvar”
