Objetivo¶

Caso de uso extensor que possibilitar vincular um componente a uma matriz curricular.

Atores¶

Principais¶

Administrador: associa um componente a uma matriz curricular.

Interessado¶

Não há.

Pré-condições¶

Pós-condições¶

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado a partir do passo FA6.3 do caso de uso UC 302 - Gerir Matrizes Curriculares <v0.1>, ao acionar a opção ENSINO > Cursos, Matrizes e Componentes > Matrizes Curriculares, em seguida, selecionando a opção Ver da matriz curricular que se deseja realizar o vínculo dentre uma das matrizes curriculares disponíveis na listagem, depois selecionar a aba Componentes Curriculares

  2. O sistema exibe informações dos componentes curriculares vinculados (ver RI1 do caso de uso UC 302 - Gerir Matrizes Curriculares <v0.1>)

  3. O administrador seleciona a opção Vincular Componente

  4. O administrador informar os dados (RIN1)

  5. O administrador finaliza o caso de uso selecionando a opção Vincular Componente

  6. O caso de uso retorna para o passo do FN.2

Fluxo Alternativo¶

FA1 – Vincular componente e continuar vinculando (FN.4 )¶

  1. O administrador aciona a opção Vincular componente e continuar vinculando

  2. O caso de uso retorna para o passo FN.4

FA2 – Editar Vínculo (FN.4 )¶

  1. O caso de uso é iniciado a partir do passo FA6.3 do caso de uso UC 302 - Gerir Matrizes Curriculares <v0.1>, ao acionar a opção ENSINO > Cursos, Matrizes e Componentes > Matrizes Curriculares, em seguida, selecionando a opção Ver da matriz curricular que se deseja realizar o vínculo dentre uma das matrizes curriculares disponíveis na listagem, depois selecionar a aba Componentes Curriculares

  2. O sistema exibe informações dos componentes curriculares vinculados (ver RI1 do caso de uso UC 302 - Gerir Matrizes Curriculares <v0.1>)

  3. O administrador seleciona a opção Editar Vínculo

  4. O sistema exibe a os dados (RIN1) do vínculo preenchidos

  5. O administrador informa novos valores para os dados (RIN1)

  6. O administrador finaliza o caso de uso selecionando a opção Vincular Componente

  7. O caso de uso retorna para o passo do FN.2
