Módulo Inscrições
-----------------

Regras relacionadas
^^^^^^^^^^^^^^^^^^^

- O aluno só pode se inscrever uma vez por programa a cada período de inscrição; caso deseje mudar algum dado, considerando que 
  o período de inscrição esteja aberto, deverá realizar nova inscrição, que substituirá a anterior.
