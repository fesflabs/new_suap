Descrição
A barra de ações deve ser utilizada ao lado do título da página (h2).
Exibição
  • Success
    • suboption 1
    • suboption 2 - suboption - suboption
    • suboption 3
  • Primary

  • Action
  • Actions
    • subaction 1
    • subaction 2
    • subaction 3
  • View
  • Danger
Regras
  1. Para mostrar o conteúdo do link dentro de um popup, deve ser adicionada à tag a a classe popup.
  2. Para botões que exigam uma confirmação (confirm javascript) antes de realizar a ação, basta adicionar a classe confirm ao link. Caso queira padronizar a mensagem do confirm, adicione como valor do atributo "data-confirm" na tag a.
  3. Para lista de ações dentro de uma célula de uma tabela que já possui conteúdo, deve ser adicionada à tag ul a classe float-end.
  4. Para marcar o botão como "checado", basta adicionar à tag a a classe checked.
Código HTML
<ul class="action-bar">
    <li><a class="btn primary" href="#"> primary </a></li>
    <li class="has-child">
        <a class="btn success" href="#"> success </a>
        <ul>
            <li><a href="#"> suboption 1 </a></li>
            <li><a href="#"> suboption 2 </a></li>
            <li><a href="#"> suboption 3 </a></li>
        </ul>
    </li>
    <li><a href="#" class="btn"> default </a></li>
    <li><a class="btn default" href="#"> info </a></li>
    <li><a class="btn danger" href="#"> danger </a></li>
</ul>
