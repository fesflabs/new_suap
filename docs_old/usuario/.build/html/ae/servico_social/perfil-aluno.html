Introdução¶

Finalidade¶

Permite preencher a Caracterização Socioeconômica e realizar inscrição para os programas do Serviço Social.

Módulo Caracterização Socioeconômica¶

Preenche/Edita Questionário¶

  1. Acesse Atividades Estudantis > Serviço Social > Caracterização Socioeconômica ou acesse Meus Dados e clique em Ações > Editar Caracterização

Módulo Inscrições¶

Regras relacionadas¶

  • O aluno só pode se inscrever uma vez por programa a cada período de inscrição; caso deseje mudar algum dado, considerando que o período de inscrição esteja aberto, deverá realizar nova inscrição, que substituirá a anterior.

Efetuar Inscrição¶

  1. Acesse Atividades Estudantis > Serviço Social > Inscrições

  2. Clique em “Efetuar Inscrição”

  3. Em Identificação (ver tela abaixo), preencha o formulário com os dados necessários:

  4. Clique em “Enviar Dados”

  5. Em Caracterização, confira os dados do aluno e clique em “Confirmar”

  6. Em Detalhamento (ver tela abaixo), preencha o formulário com os dados necessários:

  7. Clique em “Enviar Dados”
