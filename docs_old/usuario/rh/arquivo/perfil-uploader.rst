.. _suap-rh-arquivo-enviar_arquivo:

Enviar Arquivo
^^^^^^^^^^^^^^

.. |upload-arquivos| image:: ../images/upload-arquivos.png

#. Acesse Gestão de Pessoas > Arquivos > Upload de Arquivos
#. A tela "Upload de Arquivos" é exibida. Procure o servidor através do campo "Selecione o Servidor" e confirme 
   a seleção através do botão "Enviar dados"
   |upload-arquivos|
#. A tela "Upload de Arquivos" é exibida novamente. Agora ela possui três seções: "Identificação do Servidor", "Formulário de Upload de Arquivos" e 
   "Arquivos Pendentes". 

   .. note::
      Se você desejar escolher outro servidor, acesse a opção "Selecionar Outro Servidor".

   A seção **Identificação do Servidor** mostra os dados pessoais do servidor (nome, matrícula, etc.).

   A seção **Formulário de Upload de Arquivos** é usada para procurar os arquivos que se deseja enviar. Ao pressionar 
   o botão "Procurar", uma caixa de diálogo é exibida. Procure os arquivos desejados (é possível selecionar múltiplos 
   arquivos) e confirme (os arquivos serão enviados imediatamente após a confirmação).
   
   A seção **Arquivos Pendentes** mostra os arquivos que foram enviados e que ainda não foram tratados (identificados e validados), 
   ou seja, estão pendentes. 
   
     .. note::
        É possível visualizar e remover arquivos pendentes.
