Início do Documento

Objetivo¶

Cadastrar, alterar, remover e listar Horário do Campus, bem como possibilita a inclusão/remoção horários de aulas.

Atores¶

Principais¶

Administrador: permite gerir os horários dos campi.

Interessado¶

Não se aplica.

Pré-condições¶

Pós-condições¶

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado acionando a opção ENSINO > Procedimentos de Apoio > Horário do Campus

  2. O sistema exibe a lista de Horários dos Campi (RIN1)

  3. O administrador seleciona a opção Adicionar ``Horário do Campus``

  4. O administrador informa os dados (RIN2) para identificar o Horário do Campus

  5. Para cada Horários de Aulas que se deseja adicionar, o administrador informar os dados (RIN3)

  6. O administrador finaliza o caso de uso selecionando a opção Salvar

  7. O sistema exibe a mensagem M1

  8. O sistema apresenta a listagem do passo FN.2

Fluxo Alternativo¶

FA1 – Editar (FN.2 )¶

  1. O administrador aciona a opção Editar dentre um dos horários dos campi disponíveis na listagem

  2. O sistema exibe o Horário do Campus com os dados (RIN2 e RIN3) preenchidos

  3. O administrador informa novos valores para os dados (RIN2)

  4. Para cada Horários de Aulas que se deseja alterar, o administrador informar novos valores para os dados (RIN3)

  5. O administrador finaliza o caso de uso selecionando a opção Salvar

  6. O sistema exibe a mensagem M2.

  7. O sistema apresenta a listagem do passo FN.2

FA2 - Salvar e adicionar outro(a) (FN.5)¶

  1. O administrador aciona a opção Salvar e adicionar outro(a)

  2. O sistema exibe a mensagem M4.

  3. O caso de uso volta para o passo FN.5

FA3 - Salvar e continuar editando (FA1.4)¶

  1. O administrador aciona a opção Salvar e continuar editando

  2. O sistema exibe a mensagem M5.

  3. O caso de uso volta para o passo FA1.4

FA4 – Listar (FN.2)¶

  1. O administrador restringe a lista usando o filtro e/ou busca (RIN1)

  2. O sistema exibe uma lista que satisfaça aos critérios informados no passo anterior

FA5 - Remover um horário de aula (FA1.2)¶

  1. Para cada horário de aula que se deseja remover, o administrador marca a caixa de seleção Apagar?

  2. O caso de uso continua a partir do passo FA1.5

FA6 - Adicionar outro(a) Horário De Aula (FA1.2)¶

  1. O administrador aciona a opção Adicionar outro(a) Horário De Aula disponível no final da listagem Horários das Aulas para incluir uma linha vazia na listagem.

FA7 - Remover (FA1.2)¶

  1. O administrador aciona a opção Apagar

  2. O sistema exibe a mensagem M6

  3. O administrador aciona a opção “Sim, tenho certeza”

  4. O sistema exibe a mensagem M7_

  5. O administrador confirma a exclusão.

  6. O caso de uso volta para o passo FN.2

FA8 - Exportar para XLS (FN.2)¶

  1. O administrador aciona a opção Exportar para XLS

  2. O sistema faz o download do arquivo com extensão .xls com as seguintes colunas (RIN1) de acordo com a ordem existente na listagem

Requisitos de Informação¶

RIN1 – Campos para listagem de Diretoria Acadêmica¶

A listagem é exibida dividida em abas conforme especificadas abaixo:

  • Qualquer: exibe todas as diretorias acadêmicas;

  • Sem Diretores: exibe somente as diretorias acadêmicas em que não há diretores (geral, acadêmico) definidos;

  • Sem Secretários: exibe someente as diretorias acadêmicas em que não há secretários definidos;

  • Sem Coordenadores: exibe someente as diretorias acadêmicas em que não há coordenadores de cursos definidos;

  • Sem Pedagogos: exibe somente as diretorias acadêmicas em que não há pedagogos definidos.
