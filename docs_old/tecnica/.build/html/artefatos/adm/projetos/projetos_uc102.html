Objetivo¶

Cadastrar e manter atualizados os editais de projetos educacionais.

Atores¶

Principais¶

Gerente sistêmico

Interessado¶

Não se aplica.

Pré-condições¶

Não se aplica.

Pós-condições¶

Os editais estão prontos para receber projetos educacionais a partir da data determinada na definição do edital.

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado selecionando a opção EXTENSÃO > Projetos > Editais

  2. O sistema apresenta a lista de editais já cadastrados (RIN01):

  3. O ator seleciona a opção Adicionar edital

  4. Na tela Adicionar edital o ator preenche os campos (RIN02)

  5. O ator finaliza o caso de uso selecionando a opção Salvar

  6. O sistema apresenta a listagem do passo FN-2

Fluxos Alternativos¶

FA1 – Editar (FN-2)¶

  1. O ator seleciona a opção Editar de uma das linhas da listagem de editais já cadastrados

  2. Na tela Editar <nome do edital> o ator altera os campos (RIN02), os dados do edital já virão preenchidos

  3. O ator finaliza o caso de uso selecionando a opção Salvar

  4. O sistema apresenta a listagem do passo FN-2

FA2 – Salva e adicionar novo (FN-4, FA1.2)¶

  1. O ator seleciona a opção Salvar e adicionar outro(a) ao invés da opção Salvar

  2. Após salvar o edital o caso de uso reinicia no passo FN-3

FA3 – Salvar e continuar editando (FN-4, FA1.2)¶

  1. O ator seleciona a opção Salvar e continuar editando ao invés da opção Salvar

  2. Após salvar o edital o caso de uso reinicia no FA1.2

FA4 – Remover (FN-2)¶

  1. O ator seleciona a opção Remover de uma das linhas da listagem de editais já cadastrados

  2. Na tela Remover Edital o sistema apresenta uma lista todos os registros filhos relacionados ao edital

  3. O ator seleciona a opção Sim, tenho certeza para confirma que deseja remover este edital

  4. O sistema apresenta a listagem do passo FN-2

FA5 – Visualizar (FN-2)¶

  1. O ator seleciona a opção Remover de uma das linhas da listagem de editais já cadastrados

  2. Na tela Remover Edital o sistema apresenta uma lista todos os registros filhos relacionados ao edital

  3. O ator seleciona a opção Sim, tenho certeza para confirma que deseja remover este edital

  4. O sistema apresenta a listagem do passo FN-2

Fluxos de Exceção¶

FE1 – Cancelar remoção de edital (FA4.2)¶

  1. O ator seleciona a opção Cancelar na tela Remover Edital

  2. O sistema apresenta a listagem do passo FN-2
