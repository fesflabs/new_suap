Objetivo¶

Pós-condições¶

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado selecionando a opção Central de Serviços > Meus Chamados

  2. O sistema apresenta uma listagem dos chamados abertos pelo usuário ou que ele seja o interessado (RIN01):

  3. O ator seleciona a opção Visualizar Chamado

  4. Na aba Comentários o ator preenche o campo texto (RIN02)

  5. O ator finaliza o caso de uso selecionando a opção Salvar

  6. O sistema apresenta a listagem do passo FN-2
