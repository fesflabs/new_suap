Objetivo¶

Cadastra um professor convidado.

Atores¶

Principais¶

Secretário, Administrador, Coordenador, Diretor Geral e Acadêmico: podem adicionar um professor convidado.

Interessado¶

Professor.

Pré-condições¶

  1. Não pode haver mais de um professor com o mesmo CPF.

Pós-condições¶

  1. O professor como prestador de serviços é cadastrado no sistema.

  2. O professor é cadastrado no sistema.

  3. O professor é cadastrado na rede LDAP.

Casos de Uso Impactados¶

  1. UC 208 - Gerir Professor <v0.1> - Adiciona o cadastro de professores convidados.

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado acionando a opção ENSINO > Alunos e Professores > Professores

  2. O sistema exibe a lista de professores (RIN1 – Campos para listagem de professores)

  3. O Secretário seleciona a aba “Convidados”

  4. O sistema exibe uma lista de professores convidados

  5. O Secretário seleciona a opção Adicionar Professor Convidado

  6. O sistema exibe o formulário de cadastro

  7. O Secretário informa os dados (RIN1)

  8. O Secretário finaliza o caso de uso selecionando a opção Salvar

  9. O sistema exibe a mensagem M1

  10. O sistema apresenta a listagem do passo FN.2

Fluxo Alternativo¶

FA1 - Editar (FN.4)¶

  1. O Secretário aciona a opção Editar de um dos professores convidados disponíveis na listagem

  2. O sistema exibe o professor convidado com os dados (RIN1) preenchidos

  3. O Secretário informa novos valores para os dados (RIN1)

  4. O Secretário finaliza o caso de uso selecionando a opção Salvar

  5. O sistema exibe a mensagem M2.

  6. O sistema apresenta a listagem do passo FN.2

FA2 - Remover (FA1.4)¶

  1. O Secretário aciona a opção Apagar

  2. O sistema exibe a mensagem M4

  3. O Secretário aciona a opção “Sim, tenho certeza”

  4. O sistema exibe a mensagem M5

  5. O Secretário confirma a exclusão.

  6. O sistema apresenta a listagem do passo FN.2

Fluxo de Exceção¶

FE1 – Cadastro fere Regra RN2 (FN.1)¶

Exibe a mensagem M2

Regras de Negócio¶

Regra

Descrição / Mensagem

RN1

Somente professores convidados podem ser editados ou excluídos.

RN2

Os CPF são únicos, não podem haver mais de um professor convidado com o mesmo CPF.

RN3

Professores convidados com diários ou férias não podem ser excluídos.
