Início do Documento

Objetivo¶

Altera a foto do professor.

Atores¶

Principais¶

Secretário, Administrador, Coordenador, Diretor Geral e Acadêmico: podem alterar a foto do professor.

Interessado¶

Professor.

Pré-condições¶

O professor precisa estar cadastrado.

Pós-condições¶

O professor tem sua foto alterada.

Casos de Uso Impactados¶

  1. UC 208 - Gerir Professor <v0.1> - Altera a foto do professor.

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado acionando a opção ENSINO > Alunos e Professores > Professores

  2. O sistema exibe a lista de professores (RIN1 – Campos para listagem de professores)

  3. O Secretário seleciona um professor.

  4. O sistema exibe os dados gerais do professor.

  5. O Secretário seleciona a opção de edição de foto.

  6. O sistema exibe o formulário de edição de foto.

  7. O Secretário seleciona a nova foto e clica na opção Salvar.

  8. O sistema exibe a mensagem M1.

Não há.
