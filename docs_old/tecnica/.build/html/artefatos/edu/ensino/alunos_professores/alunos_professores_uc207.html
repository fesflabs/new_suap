Histórico da Revisão¶

Data

Versão

Descrição

Início do Documento

Jailton Carlos

11/05/2014

0.2

  • Exclusão da RN: exclusão de um aproveitamento de estudos muda a situação do componente para aquela anterior a situação de “Aproveitado”

  • Inclusão do FA3, FA4, M2, M3, RN5, RN6, RN7

  • Mudança do rótulo “Certificado” para “Cert. Conhecimento”

Jailton Carlos

Objetivo¶

Permite registrar ou excluir uma certificação de conhecimento um componente do histórico escolar do aluno.

Atores¶

Principais¶

Secretário, diretores acadêmicos ou administradores do sistema.

Interessado¶

Não se aplica.

Pré-condições¶

Aluno com matricula ativa

Pós-condições¶

Aluno com componente certificado

Casos de Uso Impactados¶

  1. UC 502 - Fechar Período Letivo <v0.1> : influência no cálculo no IRA

  2. UC 403 - Lançar Faltas <v0.1>: aluno desabilitado no diário para lançamento de fatlas

  3. UC 404 - Lançar Notas <v0.1>: aluno desabilitado no diário para lançamento de notas

  4. UC 300 - Manter Estrutura de curso <v0.1>:

Fluxo de Eventos¶

Fluxo Normal¶

  1. O caso de uso é iniciado a partir do passo FN.3 do caso de uso UC 200 - Visualizar Histórico <v0.1>, ao acionar a opção ENSINO > Alunos e Professores > Alunos, em seguida, selecionando a opção Ver do aluno dentre um dos alunos disponíveis na listagem, depois acionar a opção Histórico

  2. O sistema exibe o histórico (RIN1 do caso de uso UC 200 - Visualizar Histórico <v0.1>)

  3. O secretário aciona a opção Ações > Registrar Certificação de Conhecimento de um dos componente listados no histórico escolar

  4. O secretário informa os dados (RIN1)

  5. O secretário finaliza o caso de uso selecionando a opção Salvar

  6. O sistema exibe o histórico escolar (passo FN.2 ) com a situação do componente “Cert. Conhecimento”

Fluxo Alternativo¶

FA1 - Editar (FN.3)¶

  1. O sistema exibe o certificado de conhecimento com os dados (RIN1) preenchidos

  2. O secretário informa novos valores para os dados (RIN1)

  3. O secretário finaliza o caso de uso selecionando a opção Salvar

  4. O sistema exibe o histórico escolar (passo FN.2 )

FA2 - Exclusão certificação de conhecimento (FA1.1)¶

  1. O secretário aciona a opção Apagar

  2. O sistema exibe a mensagem M1

  3. o secretário confirma a exclusão

  4. O sistema exibe o histórico escolar (passo FN.2 ) com a situação do componente em aberto

FA3 - Componente com Diário (FN.5)¶

  1. O sistema exibe a mensagem M2

  2. O secretário aciona a opção “Sim, tenho certeza”

  3. O sistema exibe a mensagem M3

  4. O secretário confirma a inclusão.

  5. Continua no passo FN.6

FA4 - Aluno não obteve nota para passar (FN.5)¶

  1. O sistema exibe o histórico escolar (passo FN.2 ) com a situação do componente “Reprovado Cert. Conhecimento”

Regras de Negócio¶

Regra

Descrição / Mensagem

RN1
Critério para exibição da opção Ações > Registrar Certificação de Conhecimento: essa opção deverá está desabilitada caso:

  • O aluno tenha sido reprovado no componente curricular;

  • O aluno já tenha requerido uma certificação de conhecimento (aprovado ou não);

  • O componente não tenha sido concluído (situação aprovada, reprovada).

mensagem: não há.

RN2
Edição ou exclusão de certificação de conhecimento só será possível se ano/período letivo da certificação for maior ou igual ao ano/período letivo corrente.
Mensagem: não há.

RN3
A situação do componente só será alterada para “Cert. Conhecimento” se o estudante alcançar nota igual ou superior ao valor registrado no campo “Média para certificação de conhecimento” do caso de uso UC 300 - Manter Estrutura de curso <v0.1>
Obs.: mesmo o estudante não adquirindo nota suficiente para passar na certificação de conhecimento, o resultado da avaliação deverá ser registrado, a situação do componente será “Reprovado Cert. Conhecimento”
Mensagem: não há.

RN4
O número máximo de realização de certificação de conhecimento não deverá exceder o valor informado no campo “Número máximo de certificação de conhecimento por período” do caso de uso UC 300 - Manter Estrutura de curso <v0.1>.
Mensagem: não há.

RN5
É vedada a certificação de conhecimentos de disciplinas em que o requerente tenha sido reprovado no IFRN.
Art. 276 da Organização Didática do IFRN
Mensagem: O discente não poderá mais obter dispensa por certificação de conhecimento, pois ele foi reprovado na disciplina no período letivo anterior.

RN6
O Aluno só poderá solicitar a certificação de conhecimento, em conjunto, de até o valor informado no campo “Percentual de dispensa para certificação/aproveitamento de disciplina” do caso de uso UC 300 - Manter Estrutura de curso <v0.1> da carga horária de disciplinas do curso, para cursos técnicos de nível médio, de graduação e de pós-graduação lato sensu.
Art. 266, parágrafo único da Organização Didática do IFRN.
Mensagem: O discente não poderá mais obter dispensa por certificação de conhecimento, pois irá ultrapassar o percentual máximo permitido da carga horária de disciplinas do curso.

RN7
Para os cursos técnicos na forma integrada, é vedada a certificação de conhecimentos de disciplinas referentes aos núcleos fundamental e estruturante.
Art. 277 da Organização Didática do IFRN.
Mensagem: Para os cursos técnicos na forma integrada, é vedada a certificação de conhecimentos de disciplinas referentes aos núcleos fundamental e estruturante.
