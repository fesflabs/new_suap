É possível adicionar uma span dentro do li a com texto a ser exibido de forma sobrescrita na aba, conforme os meses Março, Setembro e Novembro acima.

Código HTML
<ul class="tabs OPTIONAL_CLASS_PARENT">
<li><a href="#">Janeiro</a></li>
<li><a href="#">Fevereiro</a></li>
<li><a href="#">Março</a></li>
</ul>
Tabela de Referência
Parâmetro Código Descrição
OPTIONAL_CLASS_PARENT .disabled Para deixar todos os itens da lista como se fossem desabilitados. Nesse caso, o li que possuir a classe active vai ser visualizado normalmente.
Regras
Para exibir o conteúdo da aba de forma integrada às abas, basta englobar cada pedaço de código com <div class="tab-container" data-titled-tab="true">, além de adicionar a class .with-content à <ul> das abas.
{% endbox %} {% box "Abas Dinâmicas" %}
Utilização

Para gerar dinamicamente as abas para os conteúdos de uma página, basta englobar cada pedaço de código com <div class="tab-container" data-title="Título">.

Atributos
Atributo Descrição
data-title Obrigatória para exibir o título da aba.
data-tab Permite utilizar a string informada aqui como parâmetro de URL.
data-counter Exibe um contador na aba.
data-checked Se True, exibe na aba. Se False, exibe .
data-hide-tab-on-counter-zero Se o atributo data-counter tiver valor igual a zero, a aba é escondida.
