Objetivo¶

Esse caso de uso descreve o preenchimento dos formuláros de avaliação por parte dos servidores.

Atores¶

  • Servidor

  • RH

Principais¶

  • Servidor

Interessado¶

  • Servidor

  • RH

Pré-condições¶

  • O Servidor terá que ser setado previamente como avaliador pelo RH, conforme foi descrito no caso de uso UC03

  • A avaliação é persistida no banco

Pós-condições¶

  • O total de pontos desta avaliação é mostrado na tela

Fluxo de Eventos¶

  • O caso de uso se inicia com uma mensagem recebida, pelo servidor, na tela de início

  • O servidor seleciona um dos servidores a serem avaliados

  • O sistema exibe o formulário de avaliação com 10 perguntas

  • O servidor preenche as 10 perguntas com notas de 0 a 10

  • O sistema vai mostrando a soma dos pontos automaticamente

  • O caso de uso é finalizado quando o servidor seleciona a opção Enviar avaliação

Fluxo Normal¶

  1. O caso de uso é iniciado acionando a opção INÍCIO

  2. O sistema exibe a mensagem de que existe servidores a serem avaliados

  3. O servidor seleciona a mensagem

  4. O sistema exibe uma lista de servidores que deverão ser avaliados

  5. O servidor seleciona um dos servidores

  6. O sistema mostra o formulário de avaliação

  7. O servidor preenche os dados de todas as notas

  8. O sistema calcula a soma das notas

  9. O servidor seleciona a opção Enviar avaliação

Fluxo Alternativo¶

FA1 – Editar avaliação (FN.1)¶

  1. O servidor aciona a opção Avaliações >> Editar

  2. O servidor seleciona uma dos servidores

  3. O sistema mostra o formulário de avaliação

  4. O servidor informa novos valores para as notas

  5. O sistema calcula a soma das notas

  6. O servidor seleciona a opção Reenviar avaliação

Fluxo de Exceção¶

FE1 – Validar campos notas (FN.1)¶

  1. No passo 7 do fluxo normal o servidor preenche um ou mais campos com nota superior a 10 ou inferior a 0

  2. O sistema exibe a mensagem: nota inválida

  3. O sistema desabilita a opção Enviar avaliação até que a nota seja corrigida

  4. O servidor corrige a nota

  5. O sistema calcula a soma das notas

  6. O sistema habilita a opção Enviar avaliação

  7. O servidor finaliza o caso de uso de execeção selecionando a opção Enviar avaliação

Especificação suplementares¶

  • O formulário de avaliação exibe perguntas diferentes quando o servidor tem função administrativa

  • O sistema precisa identificar se o servidor tem função ou não e redirecionar para o formulário de avaliação adequado
